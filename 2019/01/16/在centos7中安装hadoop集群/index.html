在centos7中安装hadoop集群

Posted on 2019-01-16

下载parcel包

网址：http://archive.cloudera.com/cdh5/parcels/latest/

文件：CDH-*-el7.parcel，CDH-*-el7.sha1

下载cm包

网址：https://archive.cloudera.com/cm6/6.1.0/redhat7/yum/RPMS/x86_64/

文件：cloudera-manager-agent-*.rpm，cloudera-manager-daemons-*.rpm，cloudera-manager-server-*.rpm

配置

关闭防火墙、SELinux

修改/etc/selinux/config文件, 将’SELINUX=enforcing’改为’SELINUX=disabled’,重启机器

设置时区

timedatectl set-timezone Asia/Shanghai

设置交换内存

/etc/sysctl.conf， 添加vm.swappiness=1

关闭透明页面压缩

echo never > /sys/kernel/mm/transparent_hugepage/defrag

echo never > /sys/kernel/mm/transparent_hugepage/enabled

主机名

master 01-20,slave 21-99

hostnamectl set-hostname master01.hadoop.com

域名

修改/etc/hosts文件，增加节点ip

NTP

安装ntp

yum install ntp

开机启动systemctl enable ntpd.service

配置ntp

ntp常用命令

安装Cloudera Manager

使用本地文件在每个节点安装

配置cloudera manager server

安装postgresql

安装server：yum --nogpgcheck localinstall -y /localpath/cloudera-manager-server-6.1.0-769885.el7.x86_64.rpm

配置数据库：/usr/share/cmf/schema/scm_prepare_database.sh -h master01.data.com postgresql scm scm scm

启动server：systemctl start cloudera-scm-server

CDH parcel包安装

登录Cloudera Manager Web

在浏览器中输入http://<server_host>:7180

Username: admin

Password: admin

创建hive，hue，oozie数据库
